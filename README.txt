# Psychologue Mayart – site statique prêt pour GitHub Pages

Fichiers à la racine (index.html, autres pages, `style.css`, logos, `CNAME`).  
Deux fichiers de redirection ont été ajoutés :
- `neurodev.html` ➜ `les-troubles-du-neurodeveloppement.html`
- `neurodeg.html` ➜ `les-maladies-neurodegeneratives.html`

Vous pouvez remplacer les images de *placeholder* par vos vraies images
(`logo-commission.png`, `logo-fwb.png`, `logo-spf.png`, `img/portrait-axel.jpg`).

Déploiement :
1) Poussez **tout le contenu** de ce dossier à la racine du dépôt GitHub.
2) Dans **Settings → Pages** : Source = `Deploy from a branch` / Branch = `main` / Folder = `/ (root)`.
3) Dans **Custom domain** : `www.psychologue-mayart.be` (le fichier `CNAME` est déjà inclus).
4) Attendez l’activation du certificat (quelques minutes à 24 h) puis cochez **Enforce HTTPS**.
